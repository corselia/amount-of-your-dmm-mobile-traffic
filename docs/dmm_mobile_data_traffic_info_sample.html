注意事項

  • 毎日午前9時以降、前日分の通信量が照会できます。
    午前9時以前の場合、前々日分を含めた30日間の履歴が表示され、午前9時以降の場合、前日分を含めた30日間の履歴が表示されます。
    例）4月15日の午前8時に確認した場合、3月15日～4月13日の期間が表示されます。
  • ご確認いただいた時点での正確な通信量を示すものではございませんので、目安としてご確認ください。
  • 日付をまたいだ連続した通信を行った場合、前日分の通信量としてカウントされる場合があります。
  • 通信量が1MB以下の場合は、小数点以下第1位までの通信量が表示されます。1MB以上は小数点は切り上げて表示されます。
  • 通信料の集計タイミング、メンテナンス時、追加通信量チャージなどの影響により、後日参照した場合に異なる場合がございます。
